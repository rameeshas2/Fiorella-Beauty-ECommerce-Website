# 📌 Project Overview

Fiorella Beauty is a premium online shopping platform created to provide customers with an engaging and user-friendly experience for discovering and purchasing beauty products.

The project focuses on:

* Modern responsive web design
* Intuitive product browsing
* Streamlined shopping experience
* Secure checkout workflow
* Professional brand identity
* Software architecture and system modeling

---

# 🎯 Project Objectives

* Create an elegant online cosmetics store
* Improve product discoverability
* Provide a responsive experience across devices
* Design an intuitive shopping journey
* Apply Software Design & Architecture concepts to a real-world project
